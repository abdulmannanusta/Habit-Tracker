# Intelligent Habit Tracker (Python)

## Overview

This project is a simple habit tracker built using Python. It helps users track their daily habits and evaluate their consistency over time.

The program allows users to add habits, mark them as completed, and view their overall progress. Based on performance, it also provides feedback using simple decision-making logic.

---

## Problem Statement

Many people start habits like studying, exercising, or reading but fail to maintain them consistently. This happens mainly due to lack of tracking and feedback.

---

## Solution

This project provides a command-line based system where users can:

* Add habits
* View habits
* Mark habits as completed
* Check progress

The system also analyzes performance and gives feedback accordingly.

---

## Features

* Add new habits
* Mark habits as completed
* Display all habits
* Calculate completion percentage
* Provide feedback based on progress

---

## AI Concepts Used

This project applies basic concepts of Artificial Intelligence:

### Decision Making

The program evaluates user performance and gives feedback such as:

* Perfect performance
* Good performance
* Needs improvement

### Rule-Based Logic

Simple conditions are used to simulate intelligent behavior:

* High progress results in positive feedback
* Low progress results in suggestions for improvement

### Basic Statistics

The program calculates completion percentage to measure consistency.
